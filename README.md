Magento2 Module Attlaz Base
============================


[![Latest Stable Version](https://img.shields.io/packagist/v/attlaz/magento2-module-attlaz-base.svg)](https://packagist.org/packages/attlaz/magento2-module-attlaz-base)

[Attlaz](https://attlaz.com) is a cloud-based integration, automation and data management platform.

## Getting started ## 

```sh
composer require attlaz/magento2-module-attlaz-base
```
